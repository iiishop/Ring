# Ring
- 这是一个用于在Windows系统中将窗口移动到指定位置的小工具
- 打开程序后，当请求完管理员权限后，即可使用左Alt键来唤醒工具。
- 按住左Alt键，程序窗口将会出现，之后使用鼠标滑动到相应的位置（上下左右、左上、右上、左下、右下）即可将窗口移动到相应的位置。
