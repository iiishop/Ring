# Ring
- 这是一个用于在Windows系统中将窗口移动到指定位置的小工具
- 打开程序后，当请求完管理员权限后，即可使用左Alt键来唤醒工具。
- 按住左Alt键，程序窗口将会出现，之后使用鼠标滑动到相应的位置（上下左右、左上、右上、左下、右下）即可将窗口移动到相应的位置。
# 版本更新
- ## 0.0.1-alpha
  - 第一个可以使用的版本，实现了基本的功能
- ## 0.0.2-alpha
  - 修复了一些bug
  - 增加了显示窗口将会被移动到什么位置的功能
- ## 0.0.3-alpha
  - 修复了一些bug
  - 提前考虑了工作区，并且窗口不再会被工作区所遮挡
  - 但窗口的移动只能在主显示器中进行（如果有多显示器），多显示器将在后续版本中支持
- ## 0.0.4-alpha
  - 支持多显示器：根据鼠标所在屏幕获取工作区，窗口吸附不再固定主显示器
  - 优化动画逻辑：移除重复延时，淡入淡出速度更稳定
  - 增加活动窗口安全判断，降低空窗口或标题读取异常导致的报错风险
  - 重构方向判断与窗口变换逻辑，减少分支判断，后续更易维护
- ## 0.0.5-alpha
  - 键盘监听回调改为通过 Tk 主线程调度 UI 更新，避免跨线程操作导致的卡顿与不稳定
  - 淡入淡出动画改为非阻塞实现，动画期间界面响应更流畅
  - 预览刷新改为按住 Alt 时才启动，松开后自动停止，减少空闲轮询
  - 提示窗界面美化：新增方向图标、层次化文字与配色，操作反馈更直观
- ## 0.0.6-alpha
  - 新增目标区域半透明预览：按住 Alt 选择方向时，实时显示窗口最终落点范围
  - 释放 Alt 后自动隐藏预览层，并执行实际吸附
- ## 0.0.7-alpha
  - 修复预览残留问题：松开 Alt 完成窗口调整后，目标区域预览会立即正确隐藏
  - UI 模块拆分：将浮层与预览相关逻辑抽离到独立 `ui.py`，主流程更清晰
  - 提示窗改为圆形视觉并移除相对坐标显示，交互反馈更简洁优雅
  - 忽略项补充：新增 `/.venv`，避免虚拟环境文件进入版本控制
- ## 0.0.8-alpha
  - 提示窗视觉重构：升级为同心圆层次化浮层，加入方向指示点与动态高亮
  - 交互反馈增强：不同吸附方向使用不同强调色，图标与文案联动更新
  - 动效升级：新增呼吸光晕动画，按住 Alt 时有更明显的操作状态反馈
  - 预览层美化：目标区域预览跟随动作强调色，整体风格更统一
- ## 0.0.9-alpha
  - 液态玻璃风格升级：重做浮层为玻璃壳体、内核、反光与光环的现代化视觉
  - 动态高光增强：加入轻微流动高光与呼吸扩散，让状态反馈更自然
  - 色彩语义强化：不同方向动作映射不同强调色，并同步到环形元素与目标预览
- ## 0.1.0-alpha
  - UI 框架迁移到 PySide6，替换原 Tkinter 实现
  - 浮层与预览层改为 Qt 自绘渲染，液态玻璃效果在动画和分层上更稳定
  - 主流程改为 Qt 事件循环，并通过信号桥接键盘监听线程，避免跨线程 UI 调用风险
- ## 0.1.1-alpha
  - 视觉风格回归大道至简：浮层仅保留极简圆环、柔和光效与方向高亮
  - 交互体验优化：动作切换时颜色与高亮角度使用平滑过渡动画
  - 预览层升级：窗口目标区域在切换方向时加入缓动形变与淡入淡出过渡
- ## 0.1.2-alpha
  - 新增布局：支持左/中/右 1/3、左/右 2/3 以及居中浮窗
  - 新增手势层级：通过拉动距离区分半屏与 1/3、2/3，保持原 8 方向手势兼容
  - 居中手势：中心区域支持直接触发居中浮窗布局
